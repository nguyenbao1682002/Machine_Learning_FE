<h1 align='center'>ESTEC Machine Learning Frontend</h1>

## **Install 📦**

1. Clone Template

```
git clone <repo-link>
```

2. Install Packages

```
yarn install
```

3. Configs environment variables

- Create a new **.env** file with same structure as **.env.example**
- Fill in your configs information in the **.env** file.

4. Start Project

```
yarn dev
```

<br/><br/>

## **Options ✍️**

1. Check lint errors or suggestions

```
yarn lint
```

2. Fix lint errors or suggestions

```
yarn lint:fix
```

3. Check prettier

```
yarn prettier
```

4. Fix prettier

```
yarn prettier:fix
```

5. Fix lint and prettier

```
yarn format
```
